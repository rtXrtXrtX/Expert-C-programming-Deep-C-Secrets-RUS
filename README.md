# Expert C Programming: Deep C Secrets — Русский перевод

Полный перевод культовой книги Питера ван дер Линдена **«Expert C Programming: Deep C Secrets»** на русский язык.

## О книге

Это не учебник для новичков. Это книга для тех, кто *уже пишет на C* и хочет понять его по-настоящему: как работают массивы и указатели на самом деле, почему `extern int *a` и `extern int a[]` — не одно и то же, что творится со стеком во время выполнения, как линкер подставляет библиотечные функции, и почему программисты путают Хэллоуин с Рождеством.

Каждая глава завершается «короткой разрядкой» — историями из мира софта, байками про разработчиков UNIX и C, фольклором хакеров Кремниевой долины и MIT.

## О переводе

Перевод выполнен с сохранением стиля, технического юмора и хакерского духа оригинала. Терминология профессиональная: «стек», «куча», «указатель», «UB», «линтить». Исходный код на C и примеры вывода терминала оставлены в оригинальном виде — нетронутыми и готовыми к компиляции.

В переводе принимали участие:
- 2oo8 (rtxrtxrtx) - инициатор перевода, C кодер 
- Вадим **hellsing** Мамуров — технический редактор
- **DeepSeek-v4** (flash) — машинный перевод с последующей литературной обработкой

## Формат

Файл в формате Markdown (.md), готовый для Obsidian, VS Code или любого другого Markdown-редактора. Исходный код обёрнут в блоки ````c ... ````, вывод терминала — в ````text ... ````.
